# 鉱産資源・電力クイズ（小4・12月）

中学受験サポート用のクイズアプリです。小4・12月の演習から、全20問を出題します。

## 出題内容
- 鉱産資源（石炭・レアメタル・都市鉱山）
- 資源の輸入先（石炭・原油・鉄鉱石・銅）とホルムズ海峡
- 発電の種類（火力・水力・原子力・地熱・風力・太陽光）
- 福島の原発事故とこれからのエネルギー
